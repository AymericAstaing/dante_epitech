# dante_epitech (TECH 1 2017-2018)

The project consists of:

- Generating perfect and imperfect mazes 
- Solving perfect and imperfect mazes

to build:

    - make / make re

to launch generator:

    ./generator x y [PERFECT]  (> .txt file)

to launch solver:

    ./solver maze.txt
